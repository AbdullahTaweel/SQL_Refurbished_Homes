# SQL_Refurbished_Homes
A database design for a platform that offers many house categories for buyers to review, select, and decorate according to personal preference.

## Articles Table
article_id	article_title	article_text	article_date	article_owner	deleted	image_id
## Products Table
product_id	product_description	product_title	product_price	product_stock	category_id
## Users
user_ID	user_name	user_surname	user_email	creationDate	security_level
## Images
image_id	image_path	image_name	image_width	image_height
